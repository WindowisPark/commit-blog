## 3개월, 15,000행의 데이터와 씨름한 결과

**KBO AI Analyzer**를 만들면서 가장 큰 도전은 "예측"이 아닌 "분석"이라는 미묘한 차이였다. 법적 포지셔닝을 위해 도박성 표현을 모두 걸러내면서도, 실제로는 61.8%의 정확도를 내는 ML 모델을 구축해야 했다.

15,026경기의 역사적 데이터에서 104개 피처를 뽑아내고, **XGBoost, ELO, LightGBM** 세 모델을 앙상블로 조합했다. 여기에 GPT-4와 Claude가 토론하는 멀티 에이전트 시스템까지 더해, 단순한 숫자 예측을 넘어선 맥락적 분석을 제공한다.

## 데이터 누수와의 끝없는 전쟁

초기 v1에서 55.7%였던 정확도가 v2에서 52.3%로 떨어진 순간이 가장 기억에 남는다. "미래 정보 누수"를 발견했기 때문이다. 경기 결과를 예측하는데 경기 당일의 데이터를 사용하고 있었던 것.

```python
# 잘못된 방식: 현재 경기 포함
team_stats = df[df['date'] <= current_date]

# 수정된 방식: 전날까지만
team_stats = df[df['date'] < current_date]
rolling_stats = team_stats.rolling(10).mean().shift(1)
```

이런 식으로 **Rolling 통계는 shift(1)**, **팀 스탯은 전년도 70% + 2년전 30% 가중 평균**으로 완전히 재구성했다. 정확도는 떨어졌지만, 실제 운영 환경에서 의미있는 수치가 됐다.

v3에서 선발투수 개인 스탯을 추가하며 59.9%로 회복했고, v4에서 홈팀 선발투수까지 포함해 **61.8%**까지 끌어올렸다. 특히 `sp_war_actual_diff` (선발투수 WAR 차이)가 가장 중요한 피처로 나타났다.

## 3명의 AI가 토론하는 시스템

단순한 ML 예측을 넘어서고 싶었다. **Analyst, Scout, Critic** 세 에이전트가 각각 다른 LLM(GPT-4o, Claude Haiku, GPT-4-turbo)을 사용해 2라운드 토론을 벌인다.

```
Phase 1: 독립 분석 ─── 3 에이전트 병렬
Phase 2: 토론 2라운드 ─ 라운드별 3 에이전트 병렬  
Phase 3: 종합 ──────── Synthesizer 1회
```

각 에이전트는 ML 모델 출력, ELO 레이팅, 최근 10경기, 선발투수 상세 정보 등을 받아 독립적으로 분석한다. Critic 에이전트는 의도적으로 반대 의견을 제시하도록 설계했다 - "Devil's Advocate" 역할로.

비용 최적화도 고려했다. Phase1 결과는 Phase2에서 이미 반영되므로 Synthesizer에는 Phase2 결과만 전달해 **토큰을 40-50% 절약**했다. 월 5경기 기준 약 54달러로 운영 가능한 구조다.

## 수익화라는 현실적 고민

기술적 완성도만으로는 지속가능하지 않다는 걸 깨달았다. **Free / Basic(₩4,900) / Pro(₩9,900)** 3단계 구독 모델을 설계하고, Stripe 결제 시스템을 연동했다.

가장 까다로운 부분은 법적 포지셔닝이었다. "적중", "픽", "배팅" 같은 도박성 표현을 모두 배제하고 "분석"이라는 용어로 일관성을 유지해야 했다. 면책 고지도 상시 노출하도록 구현했다.

티어별 콘텐츠 접근 제어는 API 레벨에서 처리한다:

```python
def filter_by_tier(prediction_result, user_tier):
    if user_tier == 'free':
        return {
            'predicted_winner': result['predicted_winner'],
            'confidence': result['confidence'],
            'home_win_probability': None,  # 숨김
            'key_factors': result['key_factors'][:1]  # 1줄만
        }
```

Free는 승리팀만, Basic은 확률까지, Pro는 에이전트 토론 전문과 ML 모델 상세까지 모두 제공한다.

## 성과와 아쉬움

3개월간 구축한 시스템의 핵심 성과:
- **ML 모델 정확도 61.8%** (무승부 제외)
- **멀티 에이전트 토론** 시스템 완성
- **완전한 수익화** 인프라 구축 
- **15,026경기 데이터** 기반 견고한 피처 엔지니어링

여전히 아쉬운 부분도 있다. 라인업 정보 수집이 KBO 공식 사이트 구조상 제한적이고, 실시간 부상/컨디션 정보는 반영하지 못한다. ELO 레이팅도 단순한 승부 결과만 반영할 뿐 경기 내용(대승, 접전 등)은 고려하지 않는다.

하지만 **기술적 완성도와 비즈니스 모델**을 동시에 갖춘 프로덕트를 만들어냈다는 점에서 의미가 있다. 특히 멀티 에이전트 토론 시스템은 단순한 확률 예측을 넘어 "왜 그렇게 생각하는지"까지 설명할 수 있는 해석 가능한 AI를 구현했다.

4월 시즌이 시작되면 실제 정확도를 검증받을 차례다. 130경기 이상에서 55% 이상의 정확도를 유지할 수 있을지가 이 시스템의 진정한 가치를 증명하는 순간이 될 것이다.

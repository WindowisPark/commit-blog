## 프로젝트 소개: 혼자가 아닌 팀과 함께하는 취업 준비

취업 준비는 외롭고 힘든 과정입니다. 단순한 챗봇으로는 한계가 있다고 느꼈고, **4명의 전문가 팀**이 함께 응답하는 멀티에이전트 서비스 JobMate를 개발하게 되었습니다.

김서연(커리어 코치), 박준호(취업 리서처), 이하은(멘탈 케어), 정민수(현직자 멘토)라는 각기 다른 성격과 전문성을 가진 AI 에이전트들이 **LangGraph** 기반으로 사용자의 감정을 분석하고, 상황에 맞게 자연스럽게 대화에 참여합니다. **GPT-4o mini**의 Function Calling을 활용해 8개의 실용적인 도구(채용검색, 이력서 피드백, 호흡운동 등)를 상황에 맞게 호출하죠.

## CSS 변수 체계화로 일관된 디자인 시스템 구축

가장 먼저 해결해야 했던 것은 흩어져 있던 컬러와 스타일링이었습니다. 하드코딩된 색상 값들을 **CSS 변수**로 체계화하여 전체 UI의 일관성을 확보했습니다.

```css
:root {
  --bg-primary: #1a1d21;
  --bg-hover: #2b2d31;
  --bg-input: #222529;
  --text-primary: #d1d2d3;
  --text-secondary: #9a9b9d;
  --accent-blue: #1264a3;
  --transition-fast: 0.15s ease;
  --radius-lg: 8px;
}
```

단순히 변수만 도입한 것이 아니라 **호버 효과**와 **트랜지션 애니메이션**을 전면적으로 개선했습니다. 메시지 버블에 마우스를 올리면 은은하게 배경이 밝아지고, 입력창에 포커스할 때 테두리 색이 부드럽게 변하는 등 미묘하지만 인터랙션을 풍부하게 만드는 디테일들을 추가했습니다.

## SVG 기반 에이전트 캐릭터 시스템

기존의 단순한 텍스트 아바타를 완전히 새롭게 디자인했습니다. 각 에이전트별로 **개성 넘치는 SVG 일러스트**를 제작하여 머리 스타일, 안경, 귀걸이 등으로 차별화했습니다.

```typescript
const AgentAvatar = ({ agentId, size }: Props) => {
  const agent = AGENTS[agentId];
  
  return (
    <div style={{
      width: size,
      height: size,
      borderRadius: 'var(--radius-lg)',
      background: `${agent.color}22`,
      border: `2px solid ${agent.color}44`,
    }}>
      <svg>{/* 에이전트별 고유 일러스트 */}</svg>
    </div>
  );
};
```

김서연은 단정한 단발머리에 진지한 표정, 박준호는 동그란 안경과 차분한 미소, 이하은은 부드러운 웨이브와 따뜻한 눈빛, 정민수는 캐주얼한 스타일로 각자의 성격이 외형에 자연스럽게 드러나도록 했습니다. 이 아바타는 채팅, 멘션 팝업, 프로필 모달, 그리고 픽셀아트 오피스까지 모든 곳에서 일관되게 사용됩니다.

## Slack 스타일 멘션 시스템 구현

실제 협업 도구처럼 자연스러운 **@멘션** 기능을 구현했습니다. 사용자가 @를 입력하면 실시간으로 에이전트 목록이 나타나고, 필터링과 키보드 네비게이션까지 지원합니다.

```typescript
const handleInputChange = (value: string) => {
  const lastAt = value.lastIndexOf("@");
  if (lastAt >= 0 && !value.slice(lastAt + 1).includes(" ")) {
    setShowMention(true);
    setMentionFilter(value.slice(lastAt + 1));
  } else {
    setShowMention(false);
  }
};
```

멘션 팝업은 단순한 이름 목록이 아니라 각 에이전트의 **프로필 사진**, **역할**, **성격**까지 표시하여 사용자가 상황에 맞는 에이전트를 쉽게 선택할 수 있게 했습니다. 또한 멘션된 텍스트는 메시지에서 하이라이트 처리되어 시각적으로도 명확하게 구분됩니다.

## 온보딩과 프로필 모달로 사용자 경험 개선

처음 서비스를 이용하는 사용자를 위한 **3단계 온보딩**을 구현했습니다. 환영 인사 → 팀 소개 → 사용법 안내 순으로 자연스럽게 서비스를 소개하고, 각 에이전트의 전문 분야와 성격을 미리 알 수 있게 했습니다.

에이전트 아바타를 클릭하면 나타나는 **프로필 모달**은 각 에이전트의 상세 정보를 보여줍니다. 전문 기술 태그, 인사말, 그리고 바로 1:1 DM으로 연결되는 버튼까지 포함하여 사용자가 각 에이전트와 더 친밀하게 소통할 수 있는 통로를 만들었습니다.

## 픽셀아트 오피스의 디테일 강화

Canvas 기반 픽셀아트 오피스도 대폭 개선했습니다. 단순한 사각형 캐릭터에서 벗어나 각 에이전트의 외형적 특징을 반영한 **스프라이트**를 제작했고, 사무용 가구, 조명 효과, 비네팅까지 추가하여 실제 사무실 같은 분위기를 연출했습니다.

에이전트들은 평소에는 자유롭게 돌아다니다가, 사용자 메시지에 응답할 때 자연스럽게 본인의 책상으로 이동해 앉아서 타이핑하는 애니메이션을 보여줍니다. 이런 디테일이 AI와의 대화를 단순한 텍스트 교환이 아닌 **실제 팀과의 협업**처럼 느끼게 만듭니다.

## 기술적 완성도와 향후 계획

이번 업데이트로 JobMate의 **UI/UX는 거의 완성**되었습니다. **React 18 + TypeScript**, **Zustand** 상태관리, **WebSocket** 실시간 통신 등 모던 프론트엔드 기술 스택을 활용하여 반응성 높고 직관적인 인터페이스를 구현했습니다.

현재는 **GPT-4o mini**의 Function Calling과 **LangGraph** 기반 멀티에이전트 시스템이 백엔드에서 안정적으로 동작하고 있으며, 8개의 실용적인 도구(채용검색, 이력서 피드백, 호흡운동 등)가 상황에 맞게 호출되고 있습니다.

다음 단계로는 **JWT 인증 시스템** 구현과 **대화 히스토리 영속화**를 통해 사용자별 맞춤형 경험을 제공할 예정입니다. 현재 메모리에만 저장되는 대화 내용을 데이터베이스에 저장하고, 이를 바탕으로 에이전트들이 사용자의 취업 준비 진행 상황을 기억하며 더욱 개인화된 조언을 제공할 수 있게 될 것입니다.
